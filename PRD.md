# 爆款菜谱AI

## 产品目标

基于《爆款菜谱》方法论生成短视频爆款文案。

用户只需要输入：

- IP定位
- 行业
- 用户群体
- 内容主题
- 视频时长
- 发布平台
- 应用场景
- 风格

系统自动输出：

- 爆款标题（10个）
- 封面文案（5个）
- 完整口播稿
- 评论区互动文案（5条）
- 爆款评分
- 优化建议

---

## 用户输入

| 字段 | 说明 |
|------|------|
| IP定位 | 创作者人设定位 |
| 行业 | 所属行业/赛道 |
| 目标用户 | 核心受众群体 |
| 内容主题 | 本期视频主题 |
| 视频时长 | 预期视频时长 |
| 平台 | 发布平台（抖音/快手/视频号等） |
| 应用场景 | 使用场景 |
| 风格 | 内容风格（口播/VLOG/剧情等） |
| 结构 | 自动推荐或手动选择（12种结构） |

---

## 12种结构

| # | 结构名称 | 结构公式 |
|---|---------|---------|
| 1 | 钩子故事型 | 钩子 + 故事 + 人设 + 总结 + 金句 |
| 2 | 问题解决型 | 问题 + 原因 + 人设 + 解决办法 + 总结 |
| 3 | 金句拆解型 | 金句 + 拆解 + 举例 + 倡导 |
| 4 | 热点观点型 | 热点 + 描述 + 观点 + 金句 |
| 5 | 利益引导型 | 利益 + 强化期待 + 方案 + 互动 |
| 6 | 揭秘故事型 | 揭秘 + 人设 + 故事 + 总结 |
| 7 | 金句轰炸型 | 金句 + 金句 + 金句 + 金句 |
| 8 | 功效展示型 | 功效 + 动作 + 效果 + 关注 |
| 9 | 简洁解决型 | 问题 + 解决办法 + 总结 |
| 10 | 现象分析型 | 现象 + 原因 + 解决办法 + 总结 |
| 11 | 钩子剖析型 | 钩子 + 剖析 + 总结 + 金句 |
| 12 | 价值堆叠型 | 价值 + 价值 + 价值 + 价值 |

---

## 输出要求

| 输出项 | 数量 | 说明 |
|--------|------|------|
| 爆款标题 | 10个 | 高点击率标题 |
| 封面文案 | 5个 | 封面配文 |
| 完整口播稿 | 1份 | 可直接录制的口播脚本 |
| 评论区互动文案 | 5条 | 引导评论互动的文案 |
| 爆款评分 | 1个 | 综合爆款潜力评分（1-100） |
| 优化建议 | 若干 | 针对性优化建议 |

---

## 交互设计

采用步骤式表单流程：

1. **基础信息** → 填写IP定位、行业、目标用户
2. **内容设定** → 选择内容主题、视频时长、发布平台
3. **场景风格** → 选择应用场景、风格
4. **结构选择** → 自动推荐3种结构 + 手动选择
5. **生成结果** → 展示完整输出 + 复制/重新生成
